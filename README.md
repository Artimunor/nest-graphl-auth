<h1>Nest GraphQL Passport Authentication</h1>

## Description

[Nest](https://github.com/nestjs/nest), [GraphQL](https://graphql.org/) & [Passport](http://www.passportjs.org/) based [Authentication](https://en.wikipedia.org/wiki/Authentication)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
